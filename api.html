API Docs

What The Domain? provides a free, white-label API for users to use. To benefit from it you simply have to make a GET request to the URL below:

https://whatthedomain.isaidpower.dev/api.php

And sending 'q' as a parameter (example: https://whatthedomain.isaidpower.dev/api.php?q=crocky). If you send the request using cURL or something similar, make sure that the request method is GET, otherwise it will be rejected.
